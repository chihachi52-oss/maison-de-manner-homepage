# LUMINA GINZA

GitHub Pagesで公開しやすい、Pyramid Resonance Sanctuary向けの静的ホームページです。

参考サイトの情報設計をもとに、文章・画像・配色・レイアウトはオリジナルで作っています。

## 変更するとよい場所

- `index.html` の `LUMINA GINZA`
- `you@example.com`
- レッスン名、料金、お知らせ、サロン住所
- `assets/salon-hero.png` を自分の写真に差し替え

## GitHub Pagesで公開

1. GitHubで新しいリポジトリを作ります。
2. この `homepage` フォルダの中身をリポジトリへアップロードします。
3. GitHubの `Settings` → `Pages` を開きます。
4. `Source` を `Deploy from a branch` にします。
5. `Branch` を `main`、フォルダを `/root` にして保存します。

数分待つと公開URLが表示されます。
